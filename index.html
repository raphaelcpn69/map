Fisca+
Connexion Google Drive
Tes notes seront synchronisées dans ton Drive — accessibles depuis n'importe quel appareil.
⚡ Cette configuration se fait une seule fois. Une fois ton Client ID enregistré, la connexion est automatique à chaque ouverture.
1
Crée un projet Google Cloud
Va sur console.cloud.google.com → clique Nouveau projet → nomme-le "Fisca+" → clique Créer.
2
Active l'API Google Drive
Dans le menu → API et services → Bibliothèque → cherche "Google Drive API" → clique Activer.
3
Configure l'écran de consentement
→ API et services → Écran de consentement OAuth → choisis Externe → remplis le nom de l'app (Fisca+) → Enregistrer.
4
Crée tes identifiants OAuth
→ Identifiants → + Créer des identifiants → ID client OAuth 2.0 → Type : Application Web.

Dans Origines JavaScript autorisées, ajoute l'URL exacte où ton fichier est hébergé (ex: https://monsite.com). Si tu testes en local, ajoute aussi http://localhost.
5
Copie ton Client ID ici
Après création, une fenêtre affiche ton ID client (se termine par .apps.googleusercontent.com). Copie-le et colle-le ci-dessous.
